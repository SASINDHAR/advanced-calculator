# Advanced Calculator

A Python calculator project with a CustomTkinter GUI, basic arithmetic helpers,
scientific expression evaluation, unit conversion utilities, and JSON-backed
calculation history.

## Project Structure

```text
advanced_calculator/
├── main.py
├── requirements.txt
├── README.md
├── calculator/
│   ├── __init__.py
│   ├── basic.py
│   ├── scientific.py
│   ├── history.py
│   ├── converter.py
│   └── utils.py
├── gui/
│   ├── __init__.py
│   ├── app.py
│   ├── themes.py
│   └── widgets.py
├── data/
│   └── history.json
└── assets/
    └── icon.png
```

## Run

```bash
pip install -r requirements.txt
python main.py
```
